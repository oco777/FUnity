# FUnity Modes（概要）

FUnity には、制作者の目的に合わせて次の 2 種類の制作モードがあります。

- **ブロックモード（Scratch モード）** — Scratch 風のステージとブロックプログラミングに特化したモード
- **unityroom モード** — unityroom への WebGL 公開を前提としたゲーム制作向けモード

主な違いや比較表は、プロジェクトルートの `Docs/Modes/README.md` を参照してください。
Unity プロジェクト内からは、`Assets/FUnity/Docs/Modes` フォルダ内の各ドキュメントを開いて確認できます。

## どちらを選ぶかの目安

- ブロックプログラミングの学習や教材用途で、Scratch に近い操作感を重視したい場合は **ブロックモード**。
- unityroom への WebGL 公開や一般的な 16:9 ゲーム画面を前提にする場合は **unityroom モード**。
